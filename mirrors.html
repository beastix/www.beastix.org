The Beastix project

Download locations - source code

Currently Beastix is available in source code form at the following locations:
Beastix at github - this is the primary location for the source tree and thus the most up to date
Beastix at gitlab - this is a secondary mirror of the base system used for gitlab CI builds

Download locations - binary builds

Currently Beastix binary builds are distributed as bootable iso images from the following locations. Please use bittorrent where available: beastix.org - this is the primary location and the most up to date - please use bittorrent
l337.org.ly - updated weekly, mirrors the latest iso build
jadedmeta.net - currently updated manually, mirrors all iso images

Becoming a mirror

If you are able to mirror either the source tree or the binary builds please get in touch at joinus@beastix.org to get listed on this page. BitTorrent seeding is also an appreciated service to help offset the bandwidth costs of hosting the binary builds - just leave your torrent client open after downloading and it will automatically seed.
