The Beastix project

About Beastix

Beastix is a project to create a unix-like operating system out of 100% free software that looks and feels like a BSD system while using the Linux kernel and various GNU components. The project is currently under development with a 0.1 release coming soon.

Where to get it

You will find the source code on github at https://github.com/beastix. Watch this space for details on the binary release once it is available.
